Exporter l'inventaire

Format ComponentHub : sauvegarde complète, réimportable telle quelle. Format Bomist : compatible avec un export/import Bomist (Stock, Part Number, Manufacturer, Description, Label, Storage, Unit Cost, Pref. Supplier, Type, Category, Worth).

Importer un fichier CSV

Le fichier doit être encodé en UTF-8. Le rapprochement se fait par identifiant (format ComponentHub) ou par référence / Part Number (format Bomist) : une correspondance met à jour l'existant, sinon un nouveau composant est créé.
